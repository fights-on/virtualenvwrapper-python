## Virtualenvwrapper-powershell

### Requirements:

* Python
* Powershell
* Virtualenv

### Installation:

```powershell
PS C:\virtualenvwrapper-powershell> python setup.py install
PS C:\> Import-Module virtualenvwrapper
```
